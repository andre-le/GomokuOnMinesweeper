# GomokuOnMinesweeper
A game which is a combination of Minesweeper and Gomoku. Its rules are the same as Gomokuâ€™s, but the game is played on Minesweeper board. When players step on the mine, they lose the X or O on that spot, and other player gets a point. The game ends either when a player reaches 5 spots a row or once the board is full, who gets more points will win.
